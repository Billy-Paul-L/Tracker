# Mon Système ⚡ — Routine Tracker

Application de suivi de routine personnelle.

## Déploiement sur Vercel

1. Fork ou upload ce repo sur GitHub
2. Connecte Vercel à GitHub
3. Crée un nouveau projet → sélectionne ce repo
4. Framework Preset: **Other** (pas de build requis)
5. Deploy !

## Utilisation sur iPhone

1. Ouvre l'URL Vercel dans **Safari**
2. Bouton Partager → "Sur l'écran d'accueil"
3. L'app s'installe comme une vraie application 📱

## Données

Les données sont sauvegardées en **localStorage** dans le navigateur — elles persistent entre les sessions sur le même appareil.
